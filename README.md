# Mac Command Agent

A simple CLI assistant to quickly search and reference Mac terminal commands.

## Features
- Topic-based command lookup
- Search commands instantly
- Extensible JSON database
- Clean CLI output

## Installation

git clone https://github.com/YOUR_USERNAME/mac-command-agent
cd mac-command-agent
pip install -r requirements.txt

## Usage

python src/main.py
python src/main.py git
python src/main.py search clone
